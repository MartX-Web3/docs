<Note>

</Note>

## Verifying contracts

All Echo Protocol contracts are verified on Etherscan. You can read the source code, ABI, and storage layout for each contract by following the Etherscan links above and clicking the **Contract** tab.

To verify a contract address programmatically, check that `eth_getCode` at the address returns non-empty bytecode on Sepolia:

```bash
cast code 0x9aeAF3881FC24A639434C4e849C52341E8b1cc15 --rpc-url $SEPOLIA_RPC_URL
```

A non-`0x` response confirms the contract is deployed at that address.
